# Basic Audio Player

A simple audio player built with HTML, CSS, and TypeScript.

## Features

- Play/Pause button
- Volume control
- Progress bar
- Time display

## How to Run

1. Open `index.html` in a web browser.
2. Replace `sample.mp3` in the HTML with your audio file path or URL.

## Development

- Compile TypeScript: `npx tsc script.ts`
- Edit `script.ts` for logic, `style.css` for styling, `index.html` for structure.
